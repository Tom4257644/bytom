From Chrome Extensions to a Full Arcade

Arcade by Tom started as a small collection of Google Chrome extensions, tiny experiments in building clean, fast, strategy-focused games you could play in a new tab. What began as a side project quickly grew into a library of simple, addictive experiences, each tuned for speed, clarity, and a little competitive edge.

Today, Arcade by Tom brings those same games to the web, with the same principles: sharp gameplay, no clutter, and a neon-clean UI that feels great on desktop or mobile. We still support the original Chrome extensions, and you can find them below.
